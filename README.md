## About MySunnyLand
SunnyLand 是一个使用 SDL3, glm, nlohmann-json 和 Tiled 开发的 C++ 跨平台平台跳跃游戏。

## WebAssembly Demo

Try the live WebAssembly demo:
- Main: [https://mjj1997.github.io/MySunnyLand/](https://mjj1997.github.io/MySunnyLand/)
- Develop: [https://mjj1997.github.io/MySunnyLand/develop/](https://mjj1997.github.io/MySunnyLand/develop/)

The `main` branch deploys to the root, `develop` to `/develop/`, and tags to `/tagname/`.
